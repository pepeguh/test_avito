# Тестовое задание по направлению frontend для Avito
## Стек 
- React, Redux Toolkit, React Router, Vite, MUI, Framer Motion, Axios, Docker 

## Установка и запуск
### 1.Docker compose
```bash
docker-compose up --build
```

### 2. Через консоль
**Backend (сервер):**
```bash
cd server
npm install
npm start
```
**Frontend:**
```bash
cd client
npm install
npm run dev
```
##**Необязательные технологии**
Framer Motion: переходы между страницами.
MUI: дизайн система.
Redux: стейт менеджер.
Axios: для упрощения работы с запросами.
dayjs: для форматирования и отображения дат.
chart.js: для создания графиков и диаграмм.
jspdf: для создания pdf отчета.
react-dom: для роутинга.
